# Android Minigames App

🇺🇸 [English](#-english) | 🇳🇱 [Nederlands](#-nederlands)

## 🇺🇸 English

This is a group project for school where we will be making an android application filled with mini-games created by us.

We used the MoSCoW method in order to prepare for this project

### Must Have:
2 Minigames per person (preferably 1 big one and one smaller one)

a Home Page displaying all the mini games.

a Navbar that let's you toggle between tabs

### Should Have:
a Credit Page to switch to

a Progress Page showing your lvl and completed games

### Could Have:
Layout/Prefrences Settings

Ability to share scores and results.

### Won't Have:
Account Database (All accounts will be saved on the local machine).

## 🇳🇱 Nederlands 

Dit is een groepsproject voor school waarbij we een Android-applicatie gaan maken met minigames die we zelf hebben bedacht.

We hebben de MoSCoW-methode gebruikt om ons op dit project voor te bereiden.

### Must Have:
2 minigames per persoon (bij voorkeur 1 grote en 1 kleinere)

Een startpagina met alle minigames.

Een navigatiebalk waarmee je tussen tabbladen kunt schakelen.

### Should Have:
Een pagina met credits om naar over te schakelen.

Een voortgangspagina met je level en voltooide games.

Layout/voorkeuren

Mogelijkheid om scores en resultaten te delen.
